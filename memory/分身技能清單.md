## 🤖 AI 工具（美編/影像幫手通用）

**AIML API 圖片生成：**
- 模型：`flux/schnell`（快速）、`flux-pro`（高品質）
- 模型：`dall-e-3`、`imagen-3.0`
- 腳本：`~/.openclaw/workspace/skills/nano-banana-2/scripts/generate_image.py`
- API Key：已寫入各分身 .env

## 📌 龍龍收到指令時的標準流程

1. **理解指令** — 少爺要我做什麼？
2. **判斷分身** — 應該找誰？
3. **派遣任務** — 用 sessions_spawn 派給分身
4. **追蹤進度** — 檢查 /tmp/subagent-results/
5. **回報結果** — 完成後稟報少爺
